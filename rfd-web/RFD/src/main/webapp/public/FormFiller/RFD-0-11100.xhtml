RFD-FF 0-11100

Special Instructions

RFD 0-11100 is a test with a combination of request parameters that are not found in common practice. The test is defined for documentation purposes but no software is included. The test software will be completed if there is a need.


Description

This test requires that the Form Filler initiate an ITI-34 transaction with the following requirements:

  1. Pre-population data is null.
  2. workflowData.formID: RFD 0-11100
  3. workflowData.encodedResponse: true, meaning the Form Filler is requesting an actual form
  4. workflowData.archiveURL: The URL of the Form Archiver in the test rig.
  5. workflowData.context: null
  6. workflowData.instanceID: null

  • The Form Filler submits the Request for Form (ITI-34) using this value for formID: RFD 0-11100
  • The test system returns a form to the Form Filler.
  • The test system evaluates the request for all required properties (both positive and negative).
